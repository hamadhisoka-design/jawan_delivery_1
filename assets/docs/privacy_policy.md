# سياسة الخصوصية

تطبيق جوان للتوصيل قد يحفظ البيانات محليًا داخل الجهاز مثل:
- الاسم
- البريد الإلكتروني
- رقم الهاتف
- الطلبات
- إشعارات داخل التطبيق
- بيانات المحفظة

لا يتم إرسال هذه البيانات إلى خوادم خارجية في النسخة الحالية ما لم يتم ربط التطبيق لاحقًا بقاعدة بيانات سحابية.

يتم استخدام البيانات فقط لتشغيل وظائف التطبيق.
